# Release Readiness Review

Check:

1. Acceptance criteria
2. Test evidence
3. Lint/type check evidence
4. Security/privacy impact
5. Backward compatibility
6. Observability/logging
7. Rollback plan
8. Operational risk
9. Documentation impact

Final output:

- SHIP / DO NOT SHIP
- Evidence
- Blockers
- Non-blocking risks
- Follow-up tasks
